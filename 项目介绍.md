# 产品洞察分析系统 — 项目介绍

## 项目简介

产品洞察分析系统是一站式电商数据智能分析平台，聚合京东、天猫、抖音、小红书四大平台数据，通过 LLM 驱动的语义分析引擎，自动生成用户评论洞察报告、品牌产品策略报告和品类趋势分析报告。

## 规格解析引擎 (shared/parse_specs.py)

三阶段提取，三平台共用：

| 阶段 | 数据来源 | 提取内容 |
|---|---|---|
| Phase 1 | 结构化属性 (attrs) | 系统、屏幕数量/类型、CPU型号、内存、硬盘、AI |
| Phase 2 | SKU 规格字符串 (color) | 配置组合 (如 `16GB / 512GB SSD`) |
| Phase 3 | 商品标题 (name) | 正则兜底，型号/CPU/内存/硬盘 |

**支持厂商**: Intel 酷睿、AMD 锐龙、高通骁龙、海思麒麟、Apple Silicon
**支持系统**: Windows、Android、macOS、iOS、Linux、HarmonyOS、ChromeOS

## 关键创新点

1. **多平台数据聚合**: 京东/Tmall/抖音/小红书统一接入
2. **AI 驱动的隐式打标**: LLM 对评论进行 8 维度语义级分析
3. **共享规格解析引擎**: JD/Tmall/Douyin 共用 `shared/parse_specs.py`
4. **服务端持久化**: SQLite 替代浏览器 localStorage，数据不丢失
5. **零运维部署**: `npm install && node server.js` 即可启动
6. **Tailwind CSS**: 现代化工具类 CSS 框架，颜色体系统一

## 启动方式

```bash
# 推荐方式
npm install
node server.js
# 访问 http://localhost:9876

# 备选方式 (PowerShell)
powershell -ExecutionPolicy Bypass -File serve.ps1

# 备选方式 (Python)
python -m http.server 9876
```

## API 端点

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | /api/proxy | AI 分析代理 (DeepSeek) |
| POST | /api/fetch-reviews | JD 评论抓取 |
| POST | /api/fetch-store-skus | JD 店铺 SKU |
| POST | /api/fetch-douyin-skus | 抖音 SKU (Apify) |
| POST | /api/fetch-tmall-skus | 天猫 SKU (Apify) |
| POST | /api/fetch-xhs-notes | 小红书笔记 |
| GET/POST | /api/settings | 用户设置 |
| GET/POST | /api/reports | 评论分析历史 |
| DELETE | /api/reports/:id | 删除评论报告 |
| GET/POST | /api/strategy-reports | 策略分析历史 |
| DELETE | /api/strategy-reports/:id | 删除策略报告 |
| GET/POST | /api/trend-reports | 趋势分析历史 |
| DELETE | /api/trend-reports/:id | 删除趋势报告 |
