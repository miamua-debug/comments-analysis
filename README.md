# 电商评论洞察分析工具

基于 AI 的电商商品评论深度分析工具。导入 CSV/Excel 评论数据，自动生成专业的多维度商品洞察报告。

## 启动方式

**方式一（推荐）**：双击 `serve.bat`，自动启动本地服务器并打开浏览器。

**方式二**：在终端中运行：
```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1
```

**方式三**：如果你有 Python：
```bash
python -m http.server 8765
# 然后打开 http://localhost:8765
```

> ⚠ **重要**：不要直接双击打开 `index.html`！浏览器出于安全策略，禁止从 `file://` 协议向 `api.anthropic.com` 发起请求，会导致 "Failed to fetch" 错误。必须通过本地 HTTP 服务器访问。

## 功能特性

- **数据导入**：支持 CSV / Excel (.xlsx/.xls) 文件拖拽或点击上传
- **智能列映射**：自动识别星级、标题、正文等关键列，支持手动调整
- **8维度深度分析**：AI 对每条评论进行人群、场景、产品、质量、服务、体验、市场、情感 8 个维度的语义提取
- **专业洞察报告**：生成包含 7 大章节的结构化报告（用户画像、核心卖点、主要痛点、改进建议、潜在机会、典型用户解析、关键洞察总结）
- **数据分析**：定量统计 + 定性语义挖掘
- **进度追踪**：实时显示分析进度，支持流式报告生成
- **报告管理**：支持复制全文、下载 TXT、历史报告存档

## 使用步骤

1. **配置 API Key**：进入「设置」页面，输入 Anthropic API Key（在 [console.anthropic.com](https://console.anthropic.com/) 获取）
2. **导入数据**：在「数据导入」页面上传评论 CSV/Excel 文件，或点击「加载示例数据」体验
3. **确认字段映射**：确认星级、标题、正文对应的列
4. **开始分析**：点击「开始智能分析」，等待 AI 生成报告
5. **查看报告**：分析完成后自动跳转到报告页，可复制或下载

## 分析框架

AI 将从以下 8 个维度对评论进行隐式打标和统计：

| 维度 | 提取内容 |
|---|---|
| 人群维度 | 性别、年龄段、职业、购买角色 |
| 场景维度 | 使用场景 |
| 产品维度 | 功能满意度、具体功能 |
| 质量维度 | 材质、做工、耐用性 |
| 服务维度 | 发货速度、包装质量、客服响应、退换货、保修 |
| 体验维度 | 舒适度、易用性、外观设计、价格感知 |
| 市场维度 | 竞品对比、复购意愿 |
| 情感维度 | 总体评价 |

## 报告章节

1. 核心用户画像与场景
2. 核心卖点与价值验证
3. 主要痛点与负面归因
4. 改进建议与优先级
5. 潜在机会与差异化
6. 典型用户深度解析
7. 关键洞察总结

## 技术架构

- 纯前端 SPA，无后端依赖
- SheetJS 库解析 Excel/CSV
- Anthropic API (Claude) 驱动 AI 分析
- localStorage 存储设置和历史
- 支持流式响应，实时展示报告生成

## 注意事项

- API Key 仅存储在浏览器本地，不会上传到任何第三方服务器
- 评论数据直接发送至 Anthropic API 进行分析，不会经过中间服务器
- 分析质量取决于评论数据的完整性和 AI 模型能力
- 推荐使用 Claude Sonnet 4.6 在成本和质量之间取得平衡
